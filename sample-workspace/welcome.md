# Welcome to the sample workspace

This is a **sample** workspace for trying out QDVC Markdown Notebook. It has
about a dozen folders, each with about a dozen notes, plus a few notes here
at the top level (which show up under **Inbox**) and a few **empty** notes
(which show up under **Empty Notes**).

## Things to try

- Browse the folders in the sidebar.
- Toggle the **Headings outline** to see the structure of a note like
  `engineering/design-notes.md`.
- Switch a tab to **Preview** to see rendered markdown.
- Use **search** to find the word _backpressure_ across notes.
- Try **Sort by date** - the notes have varied modified times.

Enjoy!
